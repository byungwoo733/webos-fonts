Summary
-------
Fonts for webOS Open Source Edition

Description
-----------
This directory contains the fonts for webOS Open Source Edition.
- Miso
- Museo Sans
- Noto Sans

Miso and Museo Sans support only Latin script.
Other scripts are supported by Noto fonts.

Note
----
NotoSans directory contains the fonts from the Noto project, chosen for
webOS Open Source Edition.
